1. Benötigte Pakete installieren

```sh
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

2. Datei `.env` erstellen, Login Daten im KLARTEXT eintragen

```env
FLENS_USERNAME=user
FLENS_PASSWORD=pass
```

3. Links und Namen der gewünschten Kursen in courses.py eintragen

4. Ans HPI wechseln
